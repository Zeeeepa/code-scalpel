## Technical Architecture
 
For the technical folks, here's how it works:
 
**Stack:**
- FastMCP for protocol implementation
- Tree-sitter for multi-language parsing
- Z3 for symbolic execution
- Pydantic for validation
 
**Languages supported:**
- Python, JavaScript, TypeScript, Java, JSX, TSX
 
**How extraction works:**
1. Parse file into AST (Abstract Syntax Tree)
2. Locate requested symbol (function/class/method)
3. Resolve dependencies (imports, helper functions)
4. Extract minimal context
5. Return to AI assistant
 
**Security:**
- Code is parsed, never executed
- 100% local processing
- No data leaves your machine
