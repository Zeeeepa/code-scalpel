## âš ï¸ Common Mistakes
 
### Don't Do This
 
1. **Too many hashtags** (looks spammy)
2. **No visuals** (low engagement)
3. **Unclear CTA** (what should they do?)
4. **Ignoring replies** (kills engagement)
5. **Self-promotion in every tweet** (unfollow)
6. **Thread too long** (>15 tweets)
7. **No hook** (scrollers skip)
8. **Marketing speak** (be authentic)
