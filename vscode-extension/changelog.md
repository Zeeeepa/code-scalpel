# Change Log

All notable changes to the "code-scalpel" extension will be documented in this file.

## [1.0.1] - 2026-01-23

### Added
- Initial release
- MCP server start/stop commands
- Code analysis command for current file
- Security scan command for entire project
- Status bar integration
- Configuration options for Python path, port, tier, and license

### Features
- Integration with Code Scalpel MCP server
- Support for community, pro, and enterprise tiers
- Context menu integration for quick access
- Output channel for server logs and analysis results

## [Unreleased]

- Inline diagnostics for security issues
